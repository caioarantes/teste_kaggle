# Computer Vision Challenge

## Objective
Develop a solution to detect and quantify plant characteristics in images.

## Suggested Dataset
https://www.kaggle.com/datasets/emmarex/plantdisease

## Proposed Tasks

### Main Task
Develop an algorithm to detect and classify:
- Leaf area
- Disease symptoms

### Technical Requirements
- Use Python
- Implement at least one of the following techniques:
    - Image segmentation
    - Deep learning classification
    - Object detection

## Deliverables
- Source code (GitHub)
- Brief presentation (5-10 minutes) covering:
    - Chosen approach
    - Challenges encountered
    - Results obtained
    - Potential improvements

## Evaluation Criteria
- Code quality
- Technical foundation
- Solution creativity
- Communication skills
- Practical implementation considerations
